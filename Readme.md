
# 🎓 Liceo Informático Manuel

Sitio web institucional tipo **landing page moderna** para el *Liceo Informático Manuel*, diseñado para presentar su propuesta educativa, atraer nuevos estudiantes y facilitar el proceso de admisión.

---

## 📌 Descripción

Este proyecto consiste en una página web profesional, responsive y optimizada para brindar una experiencia clara, atractiva y funcional a padres de familia y estudiantes interesados.

El diseño combina estética institucional con una estructura orientada a conversión, destacando la identidad del colegio:  

**Cuerpo · Mente · Alma**

---

## 🚀 Características principales

- ✅ Diseño totalmente responsive (mobile-first)
- ✅ Navegación clara con header fijo
- ✅ Hero section con mensaje institucional
- ✅ Sección “Sobre nosotros” con métricas clave
- ✅ Valores institucionales (Cuerpo, Mente, Alma)
- ✅ Oferta académica por niveles
- ✅ Proceso de admisión paso a paso
- ✅ Visualización de sedes
- ✅ Galería tipo masonry
- ✅ Formulario de contacto
- ✅ Botón flotante de WhatsApp
- ✅ UI moderna con Tailwind CSS

---

## 🛠️ Tecnologías utilizadas

- **HTML5**
- **CSS3**
- **Tailwind CSS**
- **Google Fonts (Manrope & Public Sans)**
- **Material Symbols**
- **SVG (íconos personalizados)**

---
## 🎨 Personalización
Puedes adaptar fácilmente el sitio a tus necesidades:

-   ✏️ Cambiar nombre del colegio
-   🎨 Ajustar colores institucionales (Tailwind config)
-   🖼️ Reemplazar imágenes
-   📞 Actualizar datos de contacto
-   📍 Editar sedes
-   📚 Modificar oferta académica
-   🔗 Configurar enlaces (login, WhatsApp, etc.)
